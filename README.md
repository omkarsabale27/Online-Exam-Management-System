# 📚 Online Exam Management System

A **web-based Online Exam Management System** developed using **PHP, MySQL, HTML5, CSS3, JavaScript, and Bootstrap**. The system provides a secure and efficient platform for conducting online examinations, managing students, subjects, questions, and generating exam results automatically.

---

## 🚀 Features

- 🔐 Secure Admin Login
- 👨‍🎓 Student Registration & Login
- 📝 Online Examination System
- ❓ Question Management
- 📚 Subject & Exam Management
- 👥 Student Management
- 📊 Automatic Result Generation
- ⚡ Fast & User-Friendly Interface

## ⚙️ Installation

### Step 1: Install XAMPP

Download and install XAMPP.

### Step 2: Copy Project

Copy the project folder to:

```text
C:\xampp\htdocs\
```

### Step 3: Start Server

Start the following services from the XAMPP Control Panel:

- Apache
- MySQL

### Step 4: Create Database

Open your browser:

```text
http://localhost/phpmyadmin
```

Create a new database (Example: **cee_db**).

### Step 5: Import Database

Import the SQL file:

```text
database/cee_db.sql
```

### Step 6: Run the Project

Open:

```text
http://localhost/Online-Exam-Management-System/
```
