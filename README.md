# 📚 Online Exam Management System

A web-based Online Exam Management System built using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap. This project provides a secure and efficient platform for conducting online examinations, managing students, questions, and results..

🚀 Features
🔐 Secure Admin Login
👨‍🎓 Student Login & Registration
📝 Online Examination
❓ Question Management
📊 Automatic Result Generation
👥 Student Management
📚 Subject & Exam Management
⚡ Fast and User-Friendly Interface
🛠️ Technologies Used
Technology	Purpose

MySQL	Database

⚙️ Installation
Install XAMPP.
Copy the project folder into the htdocs directory.
Start Apache and MySQL.
Open phpMyAdmin.
Create a database.
Import cee_db.sql.
Open your browser and visit:
http://localhost/Online-Exam-Management-System/
